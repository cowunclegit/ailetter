# Implementation Plan: Simplify Trend List and Mark Sent Items

**Branch**: `005-simplify-trend-list` | **Date**: 2026-01-13 | **Spec**: [specs/005-simplify-trend-list/spec.md](spec.md)
**Input**: Feature specification from `/specs/005-simplify-trend-list/spec.md`

## Summary
Simplify the curator dashboard by removing week-based filters and the "AI Selected" badge. The dashboard will now automatically display all trend items from the last 4 weeks (28 days) in a single continuous list, organized by weekly subheaders. Items already included in sent newsletters will be marked as "발송완료" (Sent), and items in drafts as "작성 중" (In Draft), with warnings for selecting sent items.

## Technical Context

**Language/Version**: Node.js (Backend), React.js (Frontend)
**Primary Dependencies**: Express, Axios, MUI, rss-parser, googleapis
**Storage**: SQLite3
**Testing**: Jest
**Target Platform**: Web Browser
**Project Type**: Web application
**Performance Goals**: Dashboard initial load < 800ms
**Constraints**: Load 28 days of data at once; provide "Sent" and "In Draft" status indicators.
